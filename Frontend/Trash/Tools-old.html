N-map

Nmap allows users to scan and discover hosts on a computer network, find open ports on those hosts, and determine the services running on those ports. It uses raw IP packets to accomplish its tasks and supports a wide range of scanning techniques, including TCP connect scanning, SYN scanning, UDP scanning, and more.

Wireshark

Wireshark is a command-line network protocol analyzer that captures and analyzes network traffic. It is part of the Wireshark suite and operates in a terminal environment. Wireshark allows users to inspect captured packets, filter specific data, and export information in various formats. It is a powerful tool for network troubleshooting, security analysis, and protocol debugging. With its extensive capabilities, Wireshark provides valuable insights into network communication and aids in diagnosing issues on networks

Strings

The "strings" tool in Linux is a command-line utility used to extract printable character sequences from binary files. It searches for ASCII strings within the file, ignoring non-printable characters and binary data. This tool is helpful in analyzing and inspecting binary files, identifying human-readable content embedded within executables, libraries, and other data files. By default, it displays sequences of four or more printable characters, but options can be used to customize its behavior and output

Binwalk

Binwalk is a command-line tool for analyzing and extracting data from binary files like firmware and disk images. It scans files for embedded files and signatures to identify various file formats and structures. With its simple interface, it helps researchers, security professionals, and developers to extract and analyze embedded components within binary data easily. Binwalk is particularly useful for tasks like reverse engineering, firmware analysis, and data extraction from complex binary files.

Exif Tool

ExifTool, you can extract information such as camera settings, GPS coordinates, date and time of creation, copyright information, and other metadata embedded in multimedia files. It supports a wide range of file formats, including JPEG, TIFF, PNG, GIF, MP3, MP4, MOV, and many more.
